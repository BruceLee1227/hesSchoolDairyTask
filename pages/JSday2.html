問題一

以下變數 a, b, c, d, e, f 它們的值、型別各自為何? (請先想想看，不要用 console.log 檢查)

  • let a = "10" + "10"; 值:1010 型別:string
  • let b = "10" * "10"; 值:100 型別:number
  • let c = 10 + "10"; 值:1010 型別:string
  • let d = 10 * "10"; 值:100 型別:number
  • let e = 10 - "2"; 值:8 型別:number
  • let f = "10" - "2"; 值:8 型別:number

問題二

請問以下題項哪些是有問題的，並說明為什麼。

  • 1. let myName = "jay"; — 合法，正常宣告變數。
  • 2. var _state = true; — 合法，變數名稱可以下劃線開頭。
  • 3. let true = true;— 有問題，true 為保留字/布林字面量，不能當變數名。
  • 4. const 520Pretty = 520;— 有問題，識別字不能以數字開頭。
  • 5. const appleNum = 10; appleNum = 2; — 有問題，const 不能重新賦值。
  • 6. var ming = '小明'; var ming = '小民'; — 合法，var 可重複宣告，最後值為 '小民'。
  • 7. const ming = '小明'; const ming = '小民';— 有問題，重複宣告同名 const 會拋錯誤。
  • 8. let ming = '小明'; let ming = '小民'; — 有問題，重複用 let 宣告同名會拋錯誤。
  • 9. let ming = '小明'; ming = '小民'; — 合法，let 允許重新賦值，最後值為 '小民'。
  • 10. let $open = "芝麻開門";— 合法，$ 為有效識別字字符。
